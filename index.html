安装指南

跟随详细步骤，安全地为您的设备安装 KernelSU

1

准备工作

  • 确保设备已解锁 Bootloader
  • 下载对应设备的 KernelSU 内核
  • 准备 ADB 和 Fastboot 工具
  • 备份重要数据
2

刷入内核

fastboot flash boot kernelsu-boot.img
fastboot reboot
3

安装管理器

  • 下载 KernelSU Manager APK
  • 安装管理器应用
  • 验证 Root 权限
4

完成配置

  • 配置应用权限管理
  • 安装必要模块
  • 享受玩机乐趣

重要提醒

刷机有风险，操作需谨慎。请确保您对刷机流程有足够了解，并已做好数据备份。

常见问题解决

遇到问题不要慌，这里有最全面的解决方案

安装后无法获取 Root 权限？

检查以下几个方面：

  • 确认内核版本与设备匹配
  • 检查 SELinux 状态
  • 重新安装 KernelSU Manager
  • 查看内核日志确认模块加载状态

设备启动后卡在开机画面？

可能的解决方案：

  • 进入 Recovery 模式，清除缓存
  • 恢复原版内核
  • 检查内核与系统版本兼容性
  • 使用正确的刷写命令

某些应用检测到 Root 权限？

隐藏 Root 的方法：

  • 使用 KernelSU 的应用权限管理
  • 安装隐藏模块
  • 配置 DenyList
  • 使用系统属性修改工具

如何卸载 KernelSU？

完全卸载步骤：

  • 刷回原版内核
  • 卸载 KernelSU Manager
  • 清除相关数据目录
  • 重启设备验证
