API 文档

简单易用的RESTful API，快速开始您的IPTV项目

获取M3U8播放列表

GET /api/v1/playlist 
curl -X GET "https://api.iptv-service.com/v1/playlist" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

响应示例

{
  "status": "success",
  "data": {
    "playlist_url": "https://cdn.iptv-service.com/playlist.m3u8",
    "total_channels": 5000,
    "last_updated": "2024-08-10T12:00:00Z"
  }
}

获取频道列表

GET /api/v1/channels 
curl -X GET "https://api.iptv-service.com/v1/channels?category=sports&limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"

获取节目指南

GET /api/v1/epg/{channel_id} 
curl -X GET "https://api.iptv-service.com/v1/epg/cctv1" \
  -H "Authorization: Bearer YOUR_API_KEY"

API认证

所有API请求都需要在请求头中包含有效的API密钥：

Authorization: Bearer YOUR_API_KEY

获取API密钥

1. 注册开发者账号

2. 创建应用并获取API密钥

3. 在请求头中添加认证信息
